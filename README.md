# Gieson Gacho — Developer Portfolio

A modern, animated, cyberpunk-neon portfolio built with **React + Vite + Tailwind CSS + Framer Motion**. Fully responsive (small phones → 4K desktop).

## Sections
Hero · Skills · Projects · Work Experience · Certificates · Contact

## Tech
- **Vite** + **React** (JavaScript / JSX)
- **Tailwind CSS** (custom cyberpunk theme)
- **Framer Motion** (scroll reveals, animated mobile sidebar, lightbox)
- `react-icons` + `lucide-react`

## Getting started
```bash
npm install
npm run dev      # start dev server
npm run build    # production build → dist/
npm run preview  # preview the production build
```

## Editing content
All content lives in `src/data/` so you can update it without touching layout:
- `site.js` — name, role, tagline, socials, stats
- `skills.js` — skill groups
- `projects.js` — projects + screenshots
- `experience.js` — work timeline
- `certificates.js` — certificates (files in `public/certificates/`)
